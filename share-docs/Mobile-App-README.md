# Mobile App

Android app for [gounderpressurepa.com](https://gounderpressurepa.com).

| Item | Path |
|------|------|
| Source repo | `ups-mobile-app/` |
| Built APK | `UPS.apk` |
| GitHub | https://github.com/Nebula112002/ups-mobile-app |

## Install on a phone

1. Copy `UPS.apk` to your Android phone
2. **Settings → Security → Install unknown apps** → allow your file manager
3. Open the APK and install

## Rebuild from source

```powershell
cd ups-mobile-app
.\build-pc.ps1
```

New debug APK: `ups-mobile-app\android\app\build\outputs\apk\debug\app-debug.apk`
